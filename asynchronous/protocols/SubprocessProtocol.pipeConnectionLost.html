    Function SubprocessProtocol.pipeConnectionLost

    Called when one of the pipes communicating with the child process is closed.

    Prototype

    abstract void pipeConnectionLost(
  int fd,
  Exception exception
);

    Parameters

    NameDescription
    fd is the file descriptor that was closed.
    exception is either an exception object or $(D_KEYWORD null). The latter means a regular EOF is received, or the connection was aborted or closed by this side of the connection.
